# koobot 机器人
> A simple bot for kook channel to share message with QQ group 😎 (。-ω-)zzz

## 项目依赖
- koa 2.13.4
- koa-bodyparser 4.3.0
- koa-router 12.0.0
- axios 0.27.2
- winston 3.8.1
- typescript 4.8.2

## Windows 快速启动
1. 修改 `index.ts` 中的 QQ群号 和 go-cqhttp 的正向代理地址
2. pnpm install
3. pnpm run serve
4. 配合 go-cqhttp 即可开始使用
